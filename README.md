## Install and Build    


```sh
git clone https://github.com/ComposableFi/composable
cd composable/
sh scripts/init.sh
cargo build --release

```

-	Currency-factory   
-	Dutch-auction   
-	Lending   
-	Liquidations   
-	Oracle   
-	Ping    
-	Transaction-fee   
-	Vault    
-	Bribedao
